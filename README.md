Multi-Disturbance Train Control Simulink Model

Description

This project provides a comprehensive Simulink model for studying train control under multi-source disturbances. It includes a 17-DOF train dynamics model, various track irregularity models, crosswind excitation, and measurement noise, allowing users to select external disturbances or internal interferences. The simulation step is optimized at 0.001 seconds for high-fidelity analysis.
